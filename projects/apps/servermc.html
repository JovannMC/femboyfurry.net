ServerMC

An open source project, heavily inspired by MultiMC, that allows you to manage multiple local Minecraft servers. Create a server, choose the server software and version, change its settings, and manage it all from the app. Create profiles as pre-configured servers to get a server up and running quicker!

This project is written in TypeScript and uses Electron + React, with plans to move to another framework later on.
This is licensed under the MIT license.

NOTE: This project is still very much WIP and scuffed - don't expect much right now!
